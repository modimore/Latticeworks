Table Styling

Not all style rules used in Latticworks can meaningfully apply to table cells. Some actually cause issues with the way table cells are displayed.

Since this is a known problem, Latticeworks includes specific property resets on cell classes specifically applied to td, th, and col elements. Notable, table cells with .lw-cell-$n-$i do not resize based on media queries unless you mark the new width as !important. Tables also do not have ::before and ::after pseudo-elements attached to them.

With this in place you can use the same width-setting classes on tables, so you can have fewer CSS classes that essentially do the same thing.
